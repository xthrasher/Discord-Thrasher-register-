# Thrasher Reg Bot
> Discord kayıt botu.

![NPM Version][npm-image]

Kayıt olan kişi çıkıp tekrar girdiğinde aynı isimle tekrar kayıt ediyor.

## Nasıl kullanılır?

```sh
ayarlar.json dosyasını editleyin
```
Birisini kayıt ettiğiniz zaman otomatik olarak db'ye isim ve yaşı pushlar test ederek kullanabilirsiniz.

## Development setup

İndirdiğiniz zipin içinde ki klasörü masaüstüne veya istediğiniz bir yere çıkartın daha sonra mouse ile dosyaya sağ tıklayın powershell(komut istemcisi) penceresini burada aça basın daha sonra aşağıda verdiğim kodları yazın.

```sh
npm install
node thrasher.js
```
